# Uninstalling Cascadia OS

## One command — macOS

```bash
curl -fsSL https://raw.githubusercontent.com/zyrconlabs/Cascadia-OS/main/uninstall.sh | bash
```

## One command — Linux

```bash
curl -fsSL https://raw.githubusercontent.com/zyrconlabs/Cascadia-OS/main/uninstall.sh | bash
```

---

Removes all Cascadia OS files, services, and login agents. Nothing outside `~/cascadia-os` and `~/Library/LaunchAgents/` is touched.

Optional tools installed alongside Cascadia (Homebrew, SwiftBar, Python) are **not** removed — only Cascadia itself.

Questions? [github.com/zyrconlabs/Cascadia-OS/issues](https://github.com/zyrconlabs/Cascadia-OS/issues)
