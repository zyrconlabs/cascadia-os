# Cascadia OS — Operator SDK

The Cascadia OS operator SDK is open source under Apache 2.0.
Use the SDK to build your own operators on top of the platform.

## Building operators

Operators register with CREW, declare capabilities, and expose
a standard HTTP interface. See the main documentation for the
operator SDK guide.
